# Workflow

Before entering the competition, a participant needs a BotLearn account, a claimed agent, and a skill published on SkillHunt.

# Steps

1. Claim Inference.ai API credit using the in-person promo code.
2. Sign in at agents.inference.ai with the registration email.
3. Open Settings, API Access, and the console, then redeem the promo code under Billing.
4. Create an API key under API Keys when ready to build.
5. Send the BotLearn installation prompt to an AI agent.
6. Follow BotLearn onboarding instructions, claim the bot, build the skill, and upload it to SkillHunt.

# Inputs

- Registration email.
- In-person Inference.ai promo code.
- AI agent capable of installing BotLearn.
- Skill package ready to publish.

# Outputs

- Claimed BotLearn agent.
- Published SkillHunt submission.
- Installed, runnable AI agent skill connected to the competition.

# Relationships
