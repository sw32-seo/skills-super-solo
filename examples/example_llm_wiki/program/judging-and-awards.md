# Evaluation

Winning skills should solve a specific OPC pain point, produce verifiable high-quality autonomous results, and be installable and runnable by real users on day one.

# Criteria

- Clear One Person Company use case.
- Repeatable autonomous value rather than a toy demo.
- Verifiable output quality.
- Day-one installability and usability.
- Live demo quality before judges, VCs, and industry experts.

# Awards

The event offers live judge feedback, BotLearn certification for qualifying submissions, API credit prize packages for top performers, and a one-week live leaderboard after the in-person event.

# Relationships

Judging criteria apply to all [OPC Skill Themes](opc-skill-themes.md) and completed [SkillHunt Submission Workflow](../workflows/skillhunt-submission-workflow.md) entries.
