# Overview

BotLearn OPC Hackathon, branded as The Super Solo AI Agent Skills Hack Day Bay Area 2026, is a live build-and-demo event for AI agent skills serving One Person Company founders.

# Details

- Date: Saturday, June 20, 2026.
- Time: 1:00 PM to 7:00 PM PT.
- Venue: Inference.ai, 333 Twin Dolphin Dr, Redwood City, CA 94065.
- Host: BotLearn.ai.
- Co-hosts: Inference.ai and SSG Accelerator.

# Relationships

Participants build skills aligned with [OPC Skill Themes](../program/opc-skill-themes.md), follow the [SkillHunt Submission Workflow](../workflows/skillhunt-submission-workflow.md), and are evaluated using the [Judging And Awards](../program/judging-and-awards.md) expectations.
