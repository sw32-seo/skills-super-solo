# Role

Hackathon participants are builders creating AI agent skills for solo founders and indie operators.

# Responsibilities

- Select an OPC pain point and theme.
- Build a working agent skill with repeatable value.
- Install BotLearn, claim an agent, and publish the skill to SkillHunt.
- Demo the skill live to judges.
- Continue competing during the one-week leaderboard period.

# Needs

Participants need event logistics, platform setup steps, API credits, theme guidance, and clear judging expectations.

# Relationships

Participants attend the [Event Overview](../events/event-overview.md), choose from [OPC Skill Themes](../program/opc-skill-themes.md), execute the [SkillHunt Submission Workflow](../workflows/skillhunt-submission-workflow.md), and target [Judging And Awards](../program/judging-and-awards.md) criteria.
